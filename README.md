# 根號拾｜彌月專員試算系統

內部作業工具：依《彌月專員教戰守則（2026年版）》規則，輸入蛋糕數量與優惠條件，
自動計算訂金 30%、免運箱數、折扣、總尾款，並產生可一鍵複製給客人的四段話術。

- 系統網址：https://chenyuyang-png.github.io/genhousefullmooncake/
- 純靜態單頁（`index.html`）；訂單預設存在使用者瀏覽器 localStorage，可選配 Google Sheet 雲端同步。
- 價目調整：頁面右上「⚙ 價格與參數設定」。

## 功能

1. 初始訂購（訂金 30% 以此為準）＋ 追加訂購（不動已收訂金、滾進尾款）
2. 買 10 送 1 贈送、月子中心 95 折、公益禮盒捐款、試吃折扣、手動折扣
3. 分批取貨・出貨排程（宅配/自取、收件人/電話/地址/到貨日、自動算箱數、完成勾選）
4. 四段客人話術產生器（訂金試算／匯款資訊／訂金確認＋表單／尾款結算）
5. 多筆訂單管理（本機儲存＋JSON 備份匯入＋Google Sheet 雲端同步）

## ☁️ 雲端同步設定（Google Sheet，一次設定全門市共用）

1. 用公司 Google 帳號開一個新的 Google 試算表（名稱隨意，例：`彌月訂單總表`）。
2. 上方選單 **擴充功能 → Apps Script**，把編輯器裡原本的內容全部刪掉，
   貼上本 repo 的 [`google-apps-script.gs`](google-apps-script.gs) 全文。
3. （可選）改第一行的 `TOKEN = 'genhouse2026'` 成自己的同步密碼。
4. 右上 **部署 → 新增部署作業**：
   - 類型選 **網頁應用程式**
   - 「執行身分」選 **我**
   - 「誰可以存取」選 **任何人**
   - 按部署 → 授權（第一次會跳 Google 授權畫面，選公司帳號、按允許）
5. 複製產生的 **網頁應用程式網址**（`https://script.google.com/macros/s/…/exec`）。
6. 回到試算系統 → 「⚙ 價格與參數設定」最下方 →
   貼上網址＋同步密碼。每台要共用的電腦都貼同一組即可。

之後：
- **☁️ 上傳 Google Sheet**：把目前這筆訂單寫進 Sheet（「訂單總覽」一位一列＋「取貨排程」一批一列）。
- **↻ 清單 → ☁️ 載入**：任何一台電腦把雲端訂單抓回來繼續編輯。
- Sheet 的「資料JSON」欄位請勿手動編輯，其他欄位隨意看、隨意篩選。
- 程式更新後要重新生效：Apps Script → 部署 → 管理部署作業 → 編輯 → 版本選「新版本」→ 部署（網址不變）。
